# Terraform Infrastructure

Infrastructure-as-code definitions for the loan automation platform environments.

## Planned Modules
- Network (VPC, subnets, security groups)
- Database (PostgreSQL via RDS)
- Object storage buckets
- ECR/EKS or ECS services
- Secrets management (AWS Secrets Manager / Parameter Store)
- Monitoring and logging (CloudWatch, OpenSearch)

> Terraform configuration will be added incrementally starting in Sprint 1 once account details are finalized.
